## Instalar
````bash
composer require barryvdh/laravel-dompdf
````

## Publicar la configuracion
````bash
php artisan vendor:publish --provider="Barryvdh\DomPDF\ServiceProvider"
````


## Comando para ejecutar basico
````bash
php artisan test
````

## Comando para ejecutarlo con el pdf
````bash
.\tests\test-to-pdf.
````
